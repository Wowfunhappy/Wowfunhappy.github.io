A fix for Apple's broken weather Dashboard widget

Last updated August 12, 2020

The Weather dashboard widget that ships with Mac OS X 10.4 – 10.14 broke in 2019, when the APIs it uses went offline. Luckily, the widget is all just editable Javascript code, so I rewrote a portion of that code to use the HERE API.

If you're running a compatible OS, just download, extract, and install the new widget to fix the Dashboard. Advanced users can also add their own API keys—please see the included readme file for details.

Sadly, this widget will not work on macOS 10.15 "Catalina" or newer, because Apple completely removed the Dashboard. Sorry about that!

Chromium Legacy Downloader

Old versions of OS X would be without an up-to-date web browser if not for Chromium Legacy, a version of Chromium (aka Google Chrome) created and maintained by Bluebox which is compatible with Mac OS X 10.7 and above.

Unlike Google Chrome, Chromium Legacy does not come with a built-in updater by default, so I added one!

The updater does not run automatically, so be sure to use the "Update Google Chrome" option in the menu bar at least once every couple of weeks.

The Chromium Legacy Downloader is a small Preference Pane which will:

  • Download and install Chromium Legacy.
  • Add an "Update" option to Chromium's menu bar.
  • Download and install Google's Widevine libraries for DRM video playback.
    • Optional.
    • OS X 10.9 Mavericks only.
  • Replace the Big Sur-style Chromium app icon with a 2012-era Chrome icon.
  • Prevent graphical glitches from appearing when a Chromium window is resized.
  • Prevent Chromium's UI from jumping downwards in fullscreen mode.
  • Force Chromium to always use traditional scrollbars, to work around glitches.
  • Tell Chromium to start in "Stable Release Mode", with fewer experimental features.
  • Tell Chromium to ignore any proxies which are enabled in System Preferences.
    • You may have configured such a proxy to fix HTTPS issues in older applications. This is unnecessary for—and can cause issues in—modern Chromium, which uses its own nework stack.
  • Change some miscellaneous default settings to better align with Apple's 2013-era human interface guidelines.

Mavericks Mini Projects

Forks, custom builds, and other small projects designed for OS X 10.9 Mavericks.

According to a random sample of people who think OS X Mavericks is the best operating system in the universe, OS X Mavericks is the best operating system in the universe. Sadly, some software developers are not aware of this research, and some of them even build software which does not work on Mavericks!

Below is a set of projects designed to run on OS X 10.9 Mavericks, from custom builds of existing open source software, to original creations which did not deserve their own page.

These downloads may or may not work on versions of OS X other than Mavericks. They may be updated infrequently, or frequently and without notice. If you're running Mavericks and encounter a problem, please feel free to shoot me an email.

Web Browsing

Chromium Legacy Downloader

Bluebox maintains a version of the Chromium Web Browser (aka Google Chrome) which is compatible with Mac OS X 10.7 and above. My Preference Pane adds some additional tweaks and bug fixes, and makes it easy to install updates.

All credit for Chromium Legacy itself goes to the wonderful and talented Bluebox. I only made this installer.

Audio & Video

QuickTime 10.2 for Mavericks

Mavericks ships with QuickTime Player 10.3, which is incompatible with third-party codecs. I've modified the Mountain Lion version of QuickTime to work on Mavericks.

Download

FFusion

Allows QuickTime 10.2 to play a variety of non-Apple video formats, including 4K vidoes downloaded from Youtube.

Based on RJVB's version of FFusion, which was based on the Perian Project's version of FFusion, which was based on the original by Jérôme Cornen. Also incorporates code by MaddTheSane.

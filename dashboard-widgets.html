A fix for two of Apple's broken Dashboard widgets

The Weather dashboard widget that ships with Mac OS X 10.4 – 10.14 broke in 2019, when the service it used went offline. Less than a year later, the currency portion of the Unit Converter widget broke as well.

Luckily, Dashboard widgets are comprised mostly of editable javascript, so I rewrote a portion of that code to retrieve data from different APIs. If you're running a compatible OS, just download, extract, and install the new widgets.

Unfortunately, I have no plans to fix Apple's other Dashboard widgets. Sorry about that!

Download Weather Widget

Compatible OS X 10.4 – 10.14.

Users are highly encouraged to add a personal API key to ensure continued functionality. Please see the included Readme for details.

Download Unit Converter

 Compatible OS X 10.8 – 10.14.
